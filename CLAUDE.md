# Богатырь — игра про Русь XIII века

Одиночная 2D-игра с открытым миром на чистом HTML5 Canvas + WebAudio. Без сборки,
без зависимостей, без сервера — запускается двойным кликом по `index.html`
(или `Запустить игру.command`) в любом браузере на любой ОС.

## Файлы

- `index.html` — разметка, CSS, HUD, оверлеи (старт/победа/смерть)
- `game.js` — вся игровая логика: бой, ИИ боссов, приручение, туман войны, миникарта, контекстные меню, прокачка
- `world.js` — процедурная генерация карты 150×150 тайлов (новая при каждом запуске); дороги гарантируют проходимость: через воду кладутся мосты, через овраги — мостки
- `sprites.js` — векторная отрисовка всех персонажей/зверей/деревьев на Canvas
- `level2.js` — весь уровень 2 «Тёмный лес»: generateWorld2, таблицы L2_WILD/L2_BOSSES,
  L2AI/L2DRAW/L2ANIMDRAW (хуки, которые game.js зовёт при `level === 2`), ЯМА (yamaRef),
  NPC со скитами (L2_NPC), волны Лона (l2Waves). Открывается после победы над Калином
  (startLevel2 из startGame); смерть на ур.2 → restartLevel2 по снапшоту l2HeroSnap.
  Верховая езда (player.mount, конь/медведь) и ковка у костра (player.forged) — в game.js.
- `test-kalin.html` / `test-level2.html` (+ *-setup.js) — мини-тесты: бой с Калином и
  сразу второй уровень на 11 уровне героя
- `audio.js` — звуковой движок: ВСЕ звуки синтезируются через WebAudio.
  Пользователь явно попросил убрать системные звуки macOS («бесят») — не возвращать их.
  Фоновая атмосфера (птицы/ветер) намеренно редкая и тихая.

## Геймплей

Цель: освободить всех пленников и убить Калина-царя в замке. Врата замка заперты,
пока живы 4 воеводы (`world.gateOpen`); сквозь стены нельзя бить и стрелять
(`wallBetween`). Внутри замка при живом Калине — красный экран + тревога
(`alarmOn`, `AudioSys.alarmStart/Stop`), после его смерти — фанфары.
Боссы: Соловей-разбойник (свист гнёт деревья), Половец («мясорубка»), Росомаха
(бросок топора + ощеривание), Кощей на ковре-самолёте (зомби-притягивание).
Дикие враги по карте (таблица `WILD_ENEMIES`, 12 видов, могут повторяться):
мясо, говённый (лужи яда `pools`), леший, кот-баюн, орёл (кровотечение), фашист,
лучник, Лихо Одноглазое (неубиваемо, паралич), людоед, чудо-юдо, чёрт, сука.
12 видов зверей можно гладить/приручать — приручённые дерутся за игрока;
E гладит, при tameBonus ≥ 0.3 приручает, иначе открывает меню ближайшего объекта;
доверие показывается в %; звери иногда не даются, хищники (PREDATORS) огрызаются.
Драконы дышат огнём (DRAGON_FIRE, breatheFire): приручённые — по врагам, злые — по герою.
Удар (playerAttack) бьёт всё: предметы ломаются за 1-3 удара (smashProp, PROP_HP),
деревья валятся за 3, звери дают сдачи/роняют припасы, пленник = −15 xp, странник бьёт посохом.
Котомка `player.items` (клавиши 1-6): ягоды/гриб/щука/мёд/букет/припасы;
припасы (meat) жарятся только у костра (+25); костёр можно сложить из бревна (ПКМ по земле).
Важные события — крупно на экран через announce() (#banner), обычные — addLog (лог+тост);
первая встреча с каждым видом врага подписывается (ENCOUNTER, metKinds).
~40% врагов (prefPlayer) целятся в героя мимо питомцев; враги расталкиваются.
Пленники: child/woman(pretty)/monk. Имя героя вводится на старте (playerName, localStorage).
Меню управляется стрелками + пробел/Enter (menuMove/menuChoose); closeMenu снимает фокус
с input (иначе клавиши «съедаются»). Панели справа тают, когда герой под ними.
Смекалка `player.smek` — шанс приручения и удача; растёт от выдуманных слов и странников.
Макс. здоровье растёт: уровень (+18), пленник (+5), родник (+15, prop 'spring').
ПКМ по любому объекту — меню действий со свободным вводом слова; при открытом
меню мир на паузе. Враги за экраном показаны стрелками по краям.
Полное описание игры для игрока — в «Инструкция.md».

UX-принципы из фидбека пользователя (соблюдать при любых правках):
- каждое действие мгновенно даёт заметную обратную связь: тост по центру (#toast),
  всплывающий текст над героем (healPlayer/floater), частицы;
- герой никогда не должен застревать: толчки только через tryMove, страховка unstickPlayer;
- лог прижат к низу flex'ом, последняя строка яркая (раньше новая строка обрезалась).

## Отладка

Игра тестируется через gstack `/browse`: `goto file://...`, клик `#startBtn`,
любое состояние доступно через `$B js` (глобалы: `player`, `enemies`, `animals`,
`hostages`, `camps`, `world`, `kalinRef`, `victory`).
